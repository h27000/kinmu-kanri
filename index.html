☁️ クラウド同期

GitHub Gistを使ってスマホ・PCでデータを共有できます。
1️⃣ GitHub → Settings → Tokens でトークンを発行
2️⃣ スコープは gist にチェック
3️⃣ 発行されたトークン（ghp_〜）を下に貼り付けて「保存」
📱 スマホで同期する手順
1️⃣ 下の「コピー」ボタンを押す
2️⃣ スマホでもこのアプリを開く
3️⃣ 設定タブ→「共有コードで接続する」にペースト→「適用」
📲 スマホ用：共有コードで接続する
PCで表示した共有コードをここに貼り付けてください
